# 巷口的咖啡店

城市的一角，藏着一家不起眼的咖啡店。

店面不大，装修也很简单，几张木质桌椅，一面摆满旧书的书架，还有一只总是趴在柜台上打盹的橘猫。老板是个话不多的人，总是专注地手冲咖啡，仿佛周围的一切都与他无关。

我常去那里，点一杯拿铁，坐在靠窗的位置。

窗外的街道不算繁华，偶尔有行人匆匆走过，偶尔有落叶随风飘落。我喜欢观察这些细碎的日常：对面面包店飘出的香气，隔壁花店老板修剪花枝的专注，还有放学后孩子们追逐打闹的笑声。

这些平凡的瞬间，让我觉得生活是真实的。

有一次，我问老板：“为什么选在这里开店？”

他想了想，说：“因为这里安静。”

是啊，安静。在这个喧嚣的城市里，能找到一个安静的地方，实属不易。

咖啡店的角落里，有一本留言本。上面写满了陌生人的心事：有人写下了梦想，有人留下了祝福，也有人只是简单写了一句“今天天气很好”。

我翻到最后一页，写下：**“愿我们都能在喧嚣中，找到属于自己的安静。”**

放下笔，喝完最后一口咖啡，我推门离开。身后，橘猫翻了个身，继续它的美梦。

生活还在继续，而这家咖啡店，会一直在这里，等着下一个推门而入的人。
