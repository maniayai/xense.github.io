# 山顶的日出

凌晨四点，闹钟响起。

我没有犹豫，翻身起床。今天，我要去看日出。

山路比想象中难走，石阶上还带着夜间的露水，有些湿滑。手电筒的光只能照亮前方几步，更远的地方，是一片漆黑。

四周很安静，只有自己的脚步声和喘息声。

走了大约四十分钟，终于到达山顶。观景台上已经有了几个人，大家都在等待。

天边开始泛起鱼肚白，然后是淡淡的橙色，再然后，是漫天的霞光。

就在太阳跃出地平线的那一刻，周围响起了掌声。

不是为了谁，只是为了这一刻的美好。

我坐在石头上，看着阳光一点点照亮山谷，照亮远处的城市。早起的不适，爬山的疲惫，在这一刻都烟消云散。

**原来，早起的人，才能看到这个世界最美的样子。**

下山的路上，遇到几个正在上山的老人。他们笑着打招呼：“小伙子，看到了吗？”

“看到了。”

“值得吧？”

“值得。”

是啊，值得。为了这一刻的日出，所有的早起和奔波，都是值得的。

回到山脚，城市已经彻底苏醒。车流人海，又是忙碌的一天。

但我知道，今天的我，和昨天的我，有一点不一样。

我见过今天的日出，它藏在我的记忆里，提醒着我：**美好，总是值得付出努力的。**
