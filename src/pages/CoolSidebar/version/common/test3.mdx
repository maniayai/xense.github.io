# 15分钟完美煎牛排

## 准备食材

- 西冷或肉眼牛排 1块（约200g）
- 迷迭香 2根
- 大蒜 3瓣（带皮拍扁）
- 黄油 20g
- 橄榄油、黑胡椒、海盐 适量

## 烹饪步骤

1.  **回温与调味**
    牛排从冰箱取出，用厨房纸吸干表面水分，室温放置 **20-30分钟**。两面均匀撒上足量的黑胡椒和海盐。

2.  **热锅下油**
    铸铁锅烧到冒烟，倒入橄榄油。下牛排，**大火**煎 **30秒** 后翻面。

3.  **增香与封边**
    转中火，下入大蒜和迷迭香。加入黄油，待黄油融化起泡，用勺子不断将汁水淋在牛排上。将牛排立起，煎一下侧面的脂肪。

4.  **醒肉**
    将牛排取出，放在烤架上静置 **5分钟**。这是锁住肉汁的关键。

5.  **装盘**
    逆着肉的纹理切片，装盘。
