# 能量满满的鸡胸肉藜麦沙拉

一款低脂高蛋白的轻食沙拉，饱腹感强，营养均衡，非常适合作为健身餐或夏日简餐。

## 🛒 准备食材

### 沙拉基底
- 鸡胸肉 1块（约150g）
- 藜麦 50g
- 生菜/混合绿叶菜 1把
- 小番茄 8-10个
- 黄瓜 半根
- 牛油果 半个

### 鸡胸肉腌料
- 橄榄油 1汤匙
- 黑胡椒碎 适量
- 盐 少许
- 蒜粉 半茶匙
- 迷迭香 1根（可选）

### 🥗 油醋汁配方
- 橄榄油 3汤匙
- 苹果醋 2汤匙
- 第戎芥末酱 1茶匙
- 蜂蜜 1茶匙
- 盐、黑胡椒 少许

## 👨‍🍳 制作步骤

### 1. 处理鸡胸肉
- 鸡胸肉横向片成两片，用刀背轻轻拍打。
- 用腌料涂抹均匀，腌制 **15-20分钟**。

### 2. 煎鸡胸肉
- 中火烧热平底锅，不放油，放入腌好的鸡胸肉。
- 一面煎 **2-3分钟** 至金黄，翻面再煎 **2分钟**。
- 转小火，盖上锅盖焖 **2-3分钟**，利用余热让内部熟透且多汁。
- 取出静置 **5分钟**，然后切片。

### 3. 煮藜麦
- 藜麦用细网筛冲洗干净。
- 放入锅中，加入 **2倍** 的水（藜麦:水 = 1:2）。
- 大火烧开，转小火煮 **15分钟**，直到水分收干，藜麦呈透明状。
- 用叉子拨散，晾凉备用。

### 4. 准备蔬菜
- 生菜洗净撕成小朵，甩干水分。
- 小番茄对半切开，黄瓜切片，牛油果切丁。

### 5. 调制油醋汁
将所有油醋汁材料放入一个带盖的玻璃瓶中，**用力摇晃乳化**至混合均匀。

### 6. 组装沙拉
在大碗中依次放入生菜、藜麦、黄瓜、小番茄、牛油果和鸡胸肉片。食用前淋上油醋汁拌匀即可。

> **💡 营养信息**：约 **450千卡**，蛋白质 **35g**，碳水 **30g**，脂肪 **20g**。
