# 外酥内软的经典巧克力曲奇

这款美式巧克力曲奇外表金黄酥脆，内部却柔软湿润，每一口都能吃到融化的巧克力。

## 🛒 准备食材

| 食材 | 用量 | 室温软化 |
| :--- | :--- | :--- |
| 无盐黄油 | 113g | ✅ 是 |
| 黄砂糖 | 100g | |
| 白砂糖 | 50g | |
| 鸡蛋 | 1个 | ✅ 是 |
| 中筋面粉 | 155g | |
| 小苏打 | 2g | |
| 盐 | 2g | |
| 黑巧克力豆 | 150g | 65%以上可可含量 |

## 👨‍🍳 制作步骤

### 1. 软化黄油
黄油提前从冰箱取出，切成小块，室温放置至用手能轻松按出指印。

### 2. 打发黄油
在软化的黄油中加入黄砂糖和白砂糖，用电动打蛋器中高速打发 **3-5分钟**，直到黄油颜色变浅、体积蓬松呈羽毛状。

### 3. 加入鸡蛋
分两次加入打散的鸡蛋液，每次都要搅打至完全融合。

### 4. 混合干性材料
在另一个碗中，将面粉、小苏打和盐混合均匀。

### 5. 混合面团
将干性材料筛入黄油混合物中，用刮刀**切拌或翻拌**至**刚好看不到干粉**即可，**不要过度搅拌**，以免面粉起筋。

### 6. 加入巧克力
倒入巧克力豆，轻轻拌匀。

### 7. 冷藏松弛
用保鲜膜盖住面团，放入冰箱冷藏 **至少1小时**（最好冷藏过夜），这一步让风味更好融合。

### 8. 烘烤
- 烤箱预热至 **190°C**。
- 将面团分成 **50g/个** 的球状，放在铺了油纸的烤盘上，每个之间留足空间。
- 放入烤箱中层，烘烤 **10-12分钟**。

### 9. 冷却
出炉时曲奇还很软，**在烤盘上冷却5分钟**定型，再转移到烤架上完全冷却。

> **💡 成功关键**：烘烤时间一定要控制好，看到边缘金黄、中心看起来还有点软时就可以出炉，余热会让它继续熟成。
