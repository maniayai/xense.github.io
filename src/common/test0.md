# 红烧肉炖蛋

一道肥而不腻、入口即化的经典红烧肉，配上吸满汤汁的鸡蛋，是名副其实的“米饭杀手”。

## 🛒 准备食材

| 食材 | 用量 | 备注 |
| :--- | :--- | :--- |
| 五花肉 | 500g | 选择带皮、肥瘦相间的 |
| 鸡蛋 | 4-5个 | 煮熟去壳备用 |
| 姜片 | 5片 | |
| 葱段 | 适量 | |
| 八角 | 2个 | |
| 香叶 | 2片 | |

## 🧂 调料清单

- 冰糖 20g（炒糖色用）
- 料酒 2汤匙
- 生抽 2汤匙
- 老抽 1汤匙
- 盐 适量

## 👨‍🍳 烹饪步骤

### 1. 处理五花肉
五花肉切成 **2-3厘米** 的方块，冷水下锅，加入姜片和料酒焯水。水开后煮 **2-3分钟**，捞出用温水洗净浮沫，沥干水分备用。

### 2. 炒糖色
锅中放少许油，加入冰糖，开**小火**慢慢搅拌。待冰糖融化成枣红色并冒起密集小泡时，糖色即炒好。**注意不要炒苦**。

### 3. 煸炒上色
迅速倒入五花肉块，快速翻炒，让每块肉都均匀裹上糖色。加入姜片、八角、香叶炒出香味。

### 4. 炖煮入味
- 加入料酒、生抽、老抽翻炒均匀。
- 加入**没过肉块的热水**（一定要用热水，肉质才不会柴）。
- 大火烧开后转**小火**，盖上锅盖炖煮 **40分钟**。

### 5. 加入鸡蛋
将煮好的鸡蛋划几刀（方便入味），放入锅中，继续炖煮 **20分钟**。

### 6. 收汁出锅
尝一下味道，根据口味加盐。转**大火收汁**，期间不停翻炒防止粘锅，待汤汁浓稠包裹在肉上即可出锅。

> **💡 小贴士**：最后收汁是关键，不要离开灶台，看到汤汁冒大泡、变浓稠时立即关火。
