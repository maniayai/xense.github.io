# 雨夜的代码

窗外的雨声淅淅沥沥，敲打着玻璃，也敲打着思绪。今晚的夜格外安静，只有键盘的哒哒声和窗外的雨声交织在一起。

我坐在书桌前，屏幕的微光映在脸上。这是连续加班的第三个夜晚，项目上线在即，还有几个棘手的 bug 需要修复。咖啡已经凉了，但代码还在继续。

有时候我会想，编程这件事，究竟是理性还是感性的？说它理性，是因为每一行代码都有其逻辑，每一个函数都有其定义。说它感性，是因为当你看到自己构建的系统流畅运行时，那种成就感是无法用言语表达的。

窗外忽然划过一道闪电，照亮了整个房间。我盯着屏幕，目光停留在那行报错的代码上。雨声渐渐变大，像是在催促我快点找到答案。

就在那一瞬间，我看到了问题所在——一个看似不起眼的空指针。修改、保存、重新运行。当控制台终于输出绿色的 "SUCCESS" 时，我长舒了一口气。

雨还在下，但心里的乌云已经散去。收拾好电脑，关掉台灯。明天又是新的一天，而今晚，是属于自己的胜利。
